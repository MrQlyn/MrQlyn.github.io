Java基本功：常见类及其方法

目录

Arrays类：全是静态方法，在java.util包

Random类：在java.util包

Math类：全是静态方法 ，在java.lang包

System类： 在java.lang包

Arrays类：全是静态方法，在java.util包

copyOf()：将一个数组复制到新数组中

sort()：将一个数组进行排序，默认为升序。改变排序规则需要重新存放子元素类的comprare方法或compareTo方法

binSearch()：在已排好序数组中查找特定值

equals()：判断俩数组是否相等

asList()：将数组重构为容器（集合）

Random类：在java.util包

Random()：生成随机数

Random(long)：带种子的伪随机数（相同种子时生成伪随即数，不设置种子生成随机数）

int nextInt(int score)：随机生成一个0~score之间的int数

nextDouble()：生成0~1直接的double数

nextFloat()：生成0~1直接的float数

nextInt()：生成0~1直接的int数

nextLong()：生成0~1直接的long数

Math类：全是静态方法 ，在java.lang包

PI：Π(3.1415926)，double型

sin()：sin值

cos()：cos值

sqrt()：平方根

min()：最小值

max()：最大值

abs()：绝对值

random()：返回[0,1]之间随机数，用法：a+Math.random()*b生成[a,a+b]之间随机数

(int)(Math.random()*a)生成[0,a]之间随机数

System类： 在java.lang包

long gc()：垃圾回收

long currentTimeMillis()：返回以毫秒为单位的时间
