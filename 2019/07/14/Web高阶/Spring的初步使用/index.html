Spring的初步使用

需要导入的包：

commons-logging-1.1.1.jar
spring-aop-4.0.0.RELEASE.jar
spring-beans-4.0.0.RELEASE.jar
spring-context-4.0.0.RELEASE.jar
spring-core-4.0.0.RELEASE.jar
spring-expression-4.0.0.RELEASE.jar

一、Bean的初步使用

1、新建一个Bean类，一定要有get和set方法。

2、在类路径下新建一个xml文件

标签：对应一个Bean类

class：Bean类的全类名，设置当前标签绑定的Bean类

id：当前标签的id，唯一标识一个标签

：对应Bean类的一个属性

name：属性名

value：属性值

3、在Java程序获取Bean对象：调用getBean(String/Class):传入String类型或者Class类型获得Bean对象，String为的id

值，当xml文件只有一个绑定了Bean类时可以使用Class，多个时只能使用String
