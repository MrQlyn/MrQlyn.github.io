Web项目里的各种路径

版权声明：转载请说明去处，文章仅供学习参考 https://blog.csdn.net/qq_38487155/article/details/82498943
访问路径:
http://localhost:8080 + /项目名/Servlet路径、 /JSP路径

Servlet路径：在web.xml里的url-pattern里配置

JSP路径：以WebRoot为根路径，不需要项目名

重定向可访问其它项目路径，所以路径必需要项目名，包括JSP路径。
转发只能访问本项目，所以路径不用项目名
